🌿 Nursery Shop Management System
A full-stack web application that allows customers to browse and order plants online, with a complete admin dashboard for managing inventory and orders.

🚀 Features

🛒 Online plant ordering system
📦 Dynamic product catalog with categories and pricing
🔐 User authentication and session management
🖥️ Admin dashboard for inventory and order management
📱 Fully responsive design for mobile and desktop
✅ Form validation and secure data handling


🛠️ Tech Stack
TechnologyUsagePHPBackend server-side logicHTML5Frontend structureCSS3Styling and responsive designMySQLDatabase management

📂 Project Structure
nursery-shop-management/
├── index.php          # Home page
├── products.php       # Plant catalog
├── cart.php           # Shopping cart
├── order.php          # Order placement
├── admin/
│   ├── dashboard.php  # Admin panel
│   ├── inventory.php  # Manage plants
│   └── orders.php     # View orders
├── css/
│   └── style.css      # Stylesheet
└── db/
    └── database.sql   # Database schema

⚙️ How to Run

Install XAMPP or WAMP
Clone this repository into htdocs folder
Import db/database.sql into phpMyAdmin
Open browser → http://localhost/nursery-shop-management
